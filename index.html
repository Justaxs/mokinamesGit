Git

Git'as yra populiariausia kodo versijavimo programa.

Git komandos

  • init - sukuria git'o repozitoriją dabartiniame aplankale
  • add - prideda nurodytus failus prie commit'ui paruoštų failų (stage'ina)
  • commit -m "žinutė" - sukuria kodo versiją iš stage'intų failų ir prideda privalomą žinutę apie pakitimus (žinutė turėtų būti neilgesnė nei 50 simbolių)
  • status - nurodo informaciją apie tavo lokalios git'o repozitorijos statusą
  • push - iš lokalios repozitorijos iššsiunčia commit'us į išorinę repozitoriją
